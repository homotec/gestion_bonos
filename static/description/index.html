  • Este modulo permite realizar una gestion simple de bonos sin necesidad de vincularlos con los pedidos.
  • Una vez instalado se han de asignar los usuarios que vayan a utilizar el modulo a los grupos de usuario GBonos User y GBonos Manager.

  • El grupo GBonos User no puede borrar bonos pero si modificarlos y crearlos.

  • El grupo GBonos Manager puede realizar cualquier operacion sobre el registro de bono.

Instalacion

  • Instalacion del modulo siguiendo el procedimiento normal de instalacion en odoo.
  • Activar el modo de desarrollador en ajustes.

  • Asignar a los usuarios con los que vayamos a trabajar con el modulo, los grupos Gbonos User y/o Gbonos Manager.
